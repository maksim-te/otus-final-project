# План на выпускную работу

1. Датасет для анализа планирую взять этот
https://clickhouse.com/docs/ru/getting-started/example-datasets/covid19
2. Подниму в Яндексе Kubernetes и в нем Apache Airflow 3 (хочется посмотреть на него)
3. Напишу трансформацию датасета из п1 на Spark (будет ли это python или scala планирую решить немного позже).
В ней из исходного датасета будет на выходе пара витрин. Какие именно, опять же, планирую решить в ходе реализации.
Spark будет читать исходный датасет из Clickhouse. Сам Clickhouse разверну как Managed Service в том же Yandex Cloud.
4. Airflow будет запускать эти трансформации и обновлять их. Тут, понятное дело, датасет статичный,
поэтому результаты трансформаций будут одинаковые. Результаты трансформаций планирую сложить назад в Clickhouse.
